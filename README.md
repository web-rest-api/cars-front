# TP - Développement Web Frontend 2024-2025

Création d'une interface web pour la gestion de voitures classiques

⚠️ Important : Utilisation de VOTRE API REST
Ce projet doit être réalisé en utilisant VOTRE API REST déployée sur Render. C'est de là que nous allons récupérer les données dynamiques. Les données locales dans mock-data.js servent uniquement de référence pour comprendre la structure des données.

## Présentation générale

Dans ce TP, vous allez construire une application web frontend complète pour gérer un catalogue de voitures classiques. Cette application communiquera avec l'API REST que vous avez développée et déployée précédemment.

## Prérequis techniques

- Connaissances en HTML/CSS/JavaScript
- Compréhension des concepts asynchrones (Promises, async/await)
- Une API REST déployée et fonctionnelle
- Un éditeur de code (VS Code recommandé)
- Git installé sur votre machine
